tensorflow学习笔记1：深度学习思维、安装

发表于 2018-07-16 | 分类于 深度学习

在开始之前，我们要先了解一下深度学习的基本思路。我们不妨从这三个视频入手，用半个小时快速学习一下神经网络：

深度学习思维

  • 神经网络的结构

  • 梯度下降法

  • 反向传播算法

在这里要再强力安利一波3Blue1Brown的数学视频，他的视频就和他的宗旨一样：深入浅出、直观明了地分享数学之美。除此之外，还有他的微积分系列和线性代数系列，都非常适合初学者入门学习。

再回顾一下视频里的内容，那个神经网络分为三个部分：输入层、隐藏层和输出层。各层之间的神经元都有不同的权重相联系，一个神经元的输出是它不同权重的各个输入之和。那么，怎么样评价一个神经网络的好坏呢？用损失来表示。损失指的是应当输出的值和实际输出的值之间的差距。了解损失之后，我们下一步就要尽可能地减少损失。视频中是使用梯度下降法来减少损失，从而提高神经网络对数字的识别率。

对于我们来说，自己手写一段代码来构建一个神经网络确实是比较麻烦的。但是所幸的是，tensorflow是一个很好的深度学习框架，里面的很多内容已经设置好了，只需要自己调参数就行。比如说，视频中的梯度下降法在tensorflow中已经有现成的包，只要会表示出损失函数就足够了。

那么现在，我们就来学习tensorflow吧！

首先，我们需要了解一下tensorflow是什么。tensorflow和numpy、Scipy这些python第三方包略有不同。那些第三方包更类似于函数的集合，只要在代码的开头import一下，就可以直接像使用python内置函数一样使用第三方函数。但tensorflow更像是一种新的语言，有着自己的语法。做个类比的话，tensorflow和python之间的关系，更类似JavaScript和HTML之间的关系。

安装

我们学习的第一步是安装tensorflow。通过官方的安装教程就可以一步一步地安装成功，但是国内的下载速度较慢，推荐使用清华的镜像。这里介绍一下我的安装环境，我使用的是 Ubuntu 16.04 LTS 系统，使用Anaconda安装。

安装成功后，在你的IDE中输入下列的代码，开始我们的学习之旅！

import tensorflow as tf
a = tf.constant("Hello")
b = tf.constant(" ")
c = tf.constant("world!")
with tf.Session() as sess:
    print(sess.run(a+b+c))
