用JATOS+lab.js建立你的在线行为实验

发表于 2019-02-07 | 分类于 个人项目

新年快乐！

这篇博客是一个关于JATOS的简单教程，用JATOS和云服务器搭建在线平台，用lab.js来编写实验程序。

学习心理学的同学应该有所了解，招募志愿者做实验可能是心理学科研中最痛苦的环节。在这个过程中，你需要和实验志愿者商量实验时间和地点，提前准备好实验设备，在实验过程中指导被试怎么做实验，最后才能收集到几份来之不易的数据。所以在实验过程中，我常常会想：能不能通过远程实验的方法来减少主试的压力呢？

有的！国外有两位名叫 Kristian Lange 和 Filevich Elisa 的大佬共同开发了一个名叫JATOS的开源框架，通过这个框架能够很方便的进行在线的行为实验。这套框架可用于心理学/经济学/认知神经等等多个领域的研究，并且可以在电脑、平板、手机等等任何带浏览器的设备上运行。关于JATOS的官方英文文档，你可以点击这里来访问。在这篇博客里我会简单介绍如何用JATOS来搭建一个在线的行为实验平台。

前期准备

购买一台云服务器

因为我们是要在线进行实验，那么肯定需要提前准备一台云服务器。如果你比较了解这一块内容的话，在自己的电脑上搭建服务器的话都是没问题的。但如果你不是很熟悉这一块的话，那么比较推荐直接租一台，方便又省事。这里我以阿里云的云主机进行示范，其它平台云主机的操作方法也大概相同。

首先打开阿里云的官网，然后在“产品”——“云计算基础”中选择“云服务器 ECS”。也可以在搜索框中直接搜索“云服务器 ESC”。

打开云服务器的页面后，如果你已经买了服务器，直接点击“管理控制台”就可以进行远程控制。如果没买的话，则需要点“立即购买”。购买服务器需要自己设置配置、时长、流量等等，比较繁琐，所以这里我比较推荐直接购买学生机，价格极低，同时配置也比较低，但完全能够满足我们的日常需求了。在搜索框里面搜“学生机”即可。

在购买页面，选择购买云服务器ECS，预装环境为Ubuntu 16.04，地域随意，时长根据自己的需要设置。

如果你选择购买学生机的话，那么可以点击这里来和我组队购买，支持一下我的工作(〃∀〃)

购买好云主机后，我们再回到管理控制台的页面，点击“实例”，这里将会显示我们购买的云主机。下图展示的是我以前购买的一个过期的主机，所以在“状态”那一栏显示“已过期”。需要注意的是，我们在“IP地址”那一栏可以看到两个IP，其中一个公有一个私有，这是什么意思呢？公有IP是指，这个IP可以通过外网来进行访问，实验志愿者们访问的也是这个IP地址；而私有IP则是阿里云内部的IP，比如主机和主机之间可以通过私有IP来通讯，这样速度更快。我们只需要记住自己的公有IP即可。

一般刚开通云主机的话，需要等待3-5分钟进行初始化，等到“状态”那一栏显示为“运行中”的时候，我们就可以点击“远程连接”来进行操作。

进入登录界面，首先我们需要在 log 那一栏输入账户名 root ，然后在 password 那一栏输入自己设定的密码。当提示 “Welcome to Alibaba Cloud Elastic Compute Service !” 时，说明我们登录成功。

安装JRE

JRE是啥？简单来说，JRE是JAVA的运行环境，如果想要在电脑上运行用Java编写的程序，则需要先安装JRE。在Ubuntu的环境中安装JRE非常容易，只需要在命令行中输入如下代码（分3次输入，每次输入一行）：

sudo apt-get update
sudo apt-get upgrade
sudo apt-get install default-jre

当提示是否安装（y/n）时，输入y并按下回车键即可。

安装完毕后，我们在命令行中输入：

java -version

如果显示了3行版本信息，那么说明jre已经成功安装。如果提示未发现java，那么说明之前的安装有问题，需要重新来过。

安装Docker

Docker是什么？简单来说，Docker可以理解成一个应用的容器。打个比方，你如果需要在手机上安装QQ，你可以在QQ的官网下载软件来安装，也可以在手机自带的软件商城里面一键安装。而Docker就相当于是这个手机的软件商城。

Docker的安装过程也很简单，只需在命令行中输入以下代码（每次一行）：

sudo apt-get update
sudo apt-get upgrade
sudo apt-get install apt-transport-https ca-certificates
sudo apt-key adv --keyserver hkp://p80.pool.sks-keyservers.net:80 --recv-keys 58118E89F3A912897C070ADBF76221572C52609D
echo "deb https://apt.dockerproject.org/repo ubuntu-xenial main" | sudo tee /etc/apt/sources.list.d/docker.list
sudo apt-get update
apt-cache policy docker-engine
sudo apt-get install docker-engine

现在Docker已经安装完毕。接下来可以测试一下Docker是否正确安装：

sudo service docker start
sudo docker run hello-world

这个命令会下载一个测试图片并且在容器（container）中运行。如果容器运行并打印出相关信息，则说明安装成功。

安装JATOS

在安装好Docker之后，安装JATOS就非常容易了，只需要两行代码：

docker pull jatos/jatos:latest
docker run -d --restart=always -p 80:9000 jatos/jatos:latest

此外，我们还需要在管理控制台中设置网络策略，不然我们无法从外网访问JATOS。在云服务器管理控制台中选择“网络和安全”——“安全组”，点击“配置规则”，然后选择克隆第三个：

在协议类型那一栏选择“全部”，优先级改为“100”。

接下来，我们在浏览器中输入你的云主机公网IP，看看会出来什么呢？

我们的服务器已经安装成功啦！初始账号密码均为 admin ，输入后即可进入控制台界面。

JATOS官方已经编写好了十几个实验案例以供参考，你可以点击这里来进行查看、下载。在控制台中，我们可以点 New Study，用Html/Css/JavaScript自己来编写一个实验程序；也可以点 Import Study，把我们在其他软件中写好的实验程序打包成JATOS的格式，导入到平台上。这里我推荐使用lab.js来编写实验程序。

lab.js是一个在线的社会和认知科学研究平台。它的工作流与E-prime, Opensesame等实验软件基本相同。也就是说，如果你学过设计实验程序，那么几乎可以零成本学会使用lab.js。并且最棒的是，它可以把设计好的实验程序直接导出为JATOS的格式。在之后的博客里，我也会写一写关于lab.js的教程，如果大家对这一方面感兴趣，或者有什么其他想了解的内容，欢迎在评论区留言(〃∀〃)
